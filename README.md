🛒 Bubble Sort - E-commerce Product Sorting

📌 Project Description
This project demonstrates the use of the Bubble Sort algorithm in an E-commerce scenario to sort product prices from low to high.

---

💡 Concept Used
- Bubble Sort Algorithm
- Python Programming

---

🛍️ Real-Time Example
In online shopping platforms, products are often sorted based on price.

Example:
Input: 1200, 800, 1500, 500  
Output: 500, 800, 1200, 1500  

---

⚙️ How It Works
- Compare adjacent elements  
- Swap if needed  
- Repeat until sorted
